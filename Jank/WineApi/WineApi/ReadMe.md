# Wine API

## Docker Container

- execute `run.bat` file
- enjoy!

## Database

- open `http://localhost:8080/` for [Adminer](http://localhost:8080/) 
- Credentials:
    - System : PostgreSQL
    - Server : db
    - Username : wineAdmin
    - Password : 12345678
    - Database : WineDb
